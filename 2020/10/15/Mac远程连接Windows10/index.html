Posted on 2020-10-15 In Mac Views:
Mac使用Microsoft Remote Desktop远程连接Windows10

在尝试 TeamViewer 过程中，Mac 和 Windows10 多次互连失败，遂决定选择使用 Microsoft 自家的远程连接软件，在此记录其中几个重要步骤

  • Mac OS 上下载安装 Microsof Remote Desktop

  • Windows10 查看内网IP（Mac和Windows10都是用校园网），命令行运行 ipconfig

  • 安装 RDPWrap
    • 下载 RDPWrap，使用管理员权限运行 install.bat，运行 RDPWInst.exe 检查是否成功
    • 显示 not listening & not supported，则 下载 rdpwrap.init 替换 C:/Program File/RDP Wrapper/rdpwrap.ini （参考Issue#1163）（可以在 issue 中找对应的版本）
    • 显示 not listening & fully supported，则 下载 rfxvmt.dll 替换 C:/Windows/system32/rfxvmt.dll

    版本为 10.0.19041.84

  • 关闭防火墙，管理->程序->开启所有 Remote Desktop Server 相关程序

  • 如果仍然问题，重启电脑解决

  • 最终效果（可实现Windows和Mac的无障碍（文件）复制粘贴）

  • 所有文件下载（适用于 10.0.19041.84）
